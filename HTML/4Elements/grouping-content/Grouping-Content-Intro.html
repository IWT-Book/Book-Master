The following list items are Grouping Content Elements and their attributes

  • Address
    The address element does not have any attributes.
  • Dl
    The dl element does not have any attributes. However, most of the time if not all the time, we use dd and dt elements with dl.
  • Dt
    The dt element does not have any attributes.
  • Figcaption
    The figcaption element does not have any attributes.
  • figure
    The figure element does not have any attributes.
  • Hr
    The hr element does not have any attributes.
  • Main
    The main element does not have any attributes.
  • Li
    The li element does not have any attributes. However, most of the time if not all the time, we use <ol> or <ul> with li.
  • Ol
    Ol attributes are,
    • Reversed - Number the list backwards.
    • Start - Ordinal value of the first item
    • Type - Kind of list marker.
  • P
    The p element does not have any attributes.
  • Pre
    The pre element does not have any attributes.
  • Ul
    The ul element does not have any attributes.
  • Blockquote
    Blockquote has only one attribute,
    • Cite - Link to the source of the quotation.
  • Dd
    The dd element does not have any attributes.
  • Div
    The div element does not have any attributes.
